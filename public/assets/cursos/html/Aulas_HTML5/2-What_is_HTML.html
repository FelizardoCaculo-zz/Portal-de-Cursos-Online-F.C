O que é HTML?

HTML, é um acrónimo para HyperText Markup Language, que é usada para criar páginas e aplicações web. Vamos ver o que significa HyperText Markup Language e Página Web.


HyperText: HyperText simplesmente significa "Texto dentro de Texto". Um texto tendo outro ligado a ele, é um hypertexto.
Sempre que voce clica em um link que te leva a uma nova página, voce clicou num hypertexto.
hypertexto, é uma forma de ligar duas ou mais páginas web (Documentos HTML).


Markup Language, ou "Linguagem de Marcação" em português, é uma linguagem que o computador usa para ao layout e convenção de formatação para o texto de um documento. A Linguagem de Marcação torna o texto mais interactivo e dinámico. Ela pode virar um texto em imagens, tabelas, links, etc.


Web Page, ou Página Web, é um documento que é comummente estrito em HTML e traduzido por um navegador web(Web Browser). A página web pode ser identificada entrando numa "URL". A página web pode ser do tipo estático ou dinámico. Com a ajuda do HTML podemos apenas criar páginas web estáticas .


Um documento HTML é formado de muitas tags HTML e cada tag contem um conteudo diferente.

Descrição do exemplo HTML

‹!DOCTYPE›
==> Esta tag define o tipo de documento, ou seja, dá instruções ao navegador sobre a versão do documento HTML. 

‹html›
==> Esta tag informa o navegador que esgte é um documento HTML. o texto deentro da tag html descreve o documento web. Esta tag é o container para todos outros elementos excepto "!DOCTYPE". 

‹head›
==> Este deve ser o primeiro elemento dentro do elemento "html", ele contem os meta dados (informações sobre o documento). Deve estar fechada antes de abrir a tag "body". 

‹title›
==> Como o nome sugere, esta tag é usada para adicionar o título da página HTML que aparece no topo da janela do navegador. Deve ser colocada dentro da tag "head". 

‹body›
==> O texto dento desta tag, descreve o conteúdo do corpo da página que é visível para o utilizador final. esta tag contem o principal conteúdo do documento HTML. 

‹h1›
==> O texto dentro da tah "h1" descreve o primeiro nível de cabeçalho da página web. 

‹p›
==> O texto dentro da tag "p" descreve o parágrafo de uma página web.


Um Breve Historial do HTML

Nos anos 80, o fisico Tim Berners-Lee que era umcontratante independente na CERN, propós um sistema para a "CERN researchers". em 1989, ele escreveu um "memo" propondo uma internet baseada no sistema de hypertexto.


Tim Berners-Lee é conhecido como o pai do HTML. A primeira descrição disponível do HTML foi um documento chamado "HTML Tags" proposto por Tim no ano 1991. A última versão do HTML é o HTML5. que aprenderemos depois neste tutorial.


Versões do HTML

Desde o tempo que o HTML foi inventado, há muitas versões do HTML no mercado, A breve introdução sobre as versões do HTML são dadas abaixo:


HTML 1.0 : A primeira versão do HTML foi 1.0, que foi a versão esqueleto da linguage HTML, e foi lançada em 1991.


HTML 2.0 : Esta foi a versão seguninte, que foi lançada em 1995, e foi a versão padrão da linguagem para o design dos sites. HTML 2.0 foi capaz de suportar atributos extras, tais como: formularios baseados no carregamento de ficheiros, formulário de elementos tais como caixa de texto, opção de botão, etc.


HTML 3.2 : Esta versão foi publicada pela W3C em 1997. Esta versão foi capaz de criar tabelas e providenciar suporte para opções extras para elementos de formulários. Ela pode também suportar uma página web com equações matemáticas complexas. Tornou-se um padrão oficial para qualquer navegador até 1997. Hoje é praticamente suportada pela maioria dos navegadores


HTML 4.01 : Esta versão foi lançada em Dezembro de 1999, e ela é uma versão muito estável da linguagem HTML. Esta versão é o actual padrão oficial, e ela provê suporte adicionado para as folhas de estilo "CSS" e habilidade de script para vários elementos de multimedia.


HTML5 : é a versão mais recente da linguagem de Marcação de Hypertexto. O primeiro projecto dessa versão foi anunciado em Janeiro de 2008. Há duas maiores organizações, uma é a "W3C" (World Wide Web Consortium), e a outra é a "WHATWG (Web Hypertext Application Tecnology Working Group) que estão envolvidos no desenvolvimento da versão 5 do HTML, e ainda, está em desenvolvimento.


Características do HTML

  1. É uma linguagem muito fácil e simples. Ela pode ser facilmente entendida e modificada.
  2. É muito fácil para fazer uma apresentação efectiva com o HTML porque ela tem muitas tags de formatação.
  3. Ela é uma linguagem de marcação, assim ela provê uma forma flexível para desenhar páginas web acompanhadas com o texto.
  4. Ela facilita programadores para adicional links na página web (com a tag ancora), assimela melhora o interesse na navegação do usuário.
  5. É uma plataforma independente porque ela pode ser apresentada em qualquer plataforma como: Windows, linux e Macintosh, etc.
  6. Facilita o programador para adicionar Gráficos, Videos e Sons para a página web que a torna mais atrativa e interactiva.
  7. HTML é uma linguagem sensível a letras Maiúsculas e Minúsculas, que significa, podemos usar tags tanto maiúsculas ou minúsculas.

NOTA: É recomendado escrever todas tags em letras minúsculas para consistência, legibilidade, etc.
